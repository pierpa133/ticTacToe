## Linux/MacOs build

1. SFML is included by FetchContet in cmake
2. `mkdir build && cd build`
3. `cmake ..`
4. `make`
5. Run program from the build directory: `./bin/TicTacToe`

## Windows build

1. SFML is included by FetchContet in cmake
2. `mkdir build && cd build`
3. `cmake ..`
4. `ninja`
5. Run program from the build directory: `./bin/TicTacToe.exe`
